# Node application
This is an [Express][expressjs] web application.


## Prerequisites

- [Node][nodejs] v0.10.x


## Native Installation (OS X/Linux)


### Install Node
- Follow the instructions from the [Node][nodejs] site.


### Install the npm packages

```
$ npm install
```
or
```
$ npm i
```

### Run the app

```
$npm start
```
